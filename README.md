# Schiesskompass – Webseite

Öffentliche Webseite von **Schiesskompass**, dem datenbasierten Schiessjournal und
Schützen-Cockpit für Schweizer Sportschützen.

*Das Schweizer Taschenmesser für den Sportschützen – von einem Sportschützen.*

Live: <https://sportschuetze.github.io/>

## Inhalt

| Seite | Zweck |
|---|---|
| Start | Was Schiesskompass ist und wem es hilft |
| Katalog | Offizielle Programme, Passen, Kranz-Anforderungen und Scheiben (nur Lesen) |
| Einreichen | Vereins-Jahresprogramme und vereinsinterne Programme per E-Mail melden |
| Hilfe | Benutzerhilfe zu den wichtigsten Bereichen der App |
| Glossar | Begriffe rund um den Schweizer Schiesssport |
| Über & Kontakt | Wer dahintersteckt, Feedback und Kontakt |
| Datenschutz / Impressum | Rechtliche Angaben |

## Technik

Statische Webseite (HTML/CSS/JS, kein Build-Schritt), ausgeliefert über GitHub Pages.
Die Katalog- und Quellen-Ansicht laden `catalog.json` bzw. das Manifest live aus dem
Daten-Repository [Sportschuetze/Schiesskompass](https://github.com/Sportschuetze/Schiesskompass).
